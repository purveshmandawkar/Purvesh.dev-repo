Automated Soil Irrigation System

A closed-loop irrigation system that reads soil moisture in real time and controls a water pump through hysteresis logic — preventing both over-watering and the relay chatter that destroys simpler threshold-based designs.

Capacitive Sensing Hysteresis Control Relay Actuation Embedded Firmware Embedded C

The Problem with Simple Threshold Control

Most beginner irrigation projects use a single moisture threshold: pump ON below 50%, pump OFF above 50%. This causes rapid relay switching when moisture hovers near the threshold — the pump flickers ON and OFF continuously, burning out the relay and motor within weeks.

The solution is hysteresis — two separate thresholds with a buffer zone between them.

Hysteresis Logic — How it works

Pump ON
≤ 30%
Buffer
30–70%
Pump OFF
≥ 70%

The 40% buffer zone means the pump never switches ON and OFF rapidly. Once ON, it stays ON until moisture reaches 70%. Once OFF, it stays OFF until moisture drops to 30%. This protects both the relay and the motor.

Components

5V Relay ModuleElectrically isolates the low-power control logic from the high-power pump
DC Submersible PumpMoves water from reservoir to soil when triggered

Why capacitive over resistive sensors? Resistive sensors pass current through the soil — this causes electrolysis, which corrodes the sensor probes within weeks. Capacitive sensors measure dielectric change without direct current flow, so they last far longer in damp soil.

Irrigation Firmware — Calibrated ADC + Hysteresis Control

#define MOISTURE_PIN  34
#define RELAY_PIN     25

// Calibrated sensor range (measured in dry air and submerged)
const int DRY_VAL  = 3500;  // Raw ADC reading in dry air
const int WET_VAL  = 1500;  // Raw ADC reading fully submerged

// Hysteresis thresholds
const int MIN_THRES = 30;   // Pump ON  below this
const int MAX_THRES = 70;   // Pump OFF above this

void setup() {
  Serial.begin(115200);
  pinMode(RELAY_PIN, OUTPUT);
  digitalWrite(RELAY_PIN, HIGH);  // Start with pump OFF (active low)
}

void loop() {
  int raw = analogRead(MOISTURE_PIN);

  // Map raw ADC value to 0–100% using calibrated endpoints
  int moisture = map(raw, DRY_VAL, WET_VAL, 0, 100);
  moisture = constrain(moisture, 0, 100);  // Clamp to valid range

  // Hysteresis logic — prevents relay chatter
  if (moisture <= MIN_THRES) {
    digitalWrite(RELAY_PIN, LOW);   // Pump ON — soil is dry
  } else if (moisture >= MAX_THRES) {
    digitalWrite(RELAY_PIN, HIGH);  // Pump OFF — soil is wet enough
  }
  // Between 30–70%: do nothing — pump stays in its current state

  Serial.printf("Moisture: %d%%\n", moisture);
  delay(1000);
}

Engineering Details

ConceptImplementation
Sensor CalibrationEvery soil type has different conductivity. The map() function translates raw ADC voltage into a human-readable percentage using measured dry and wet endpoints
Active-low RelayRelay starts HIGH (OFF) on boot — safe default prevents pump from running on power-up
Hysteresis Buffer40% gap between ON and OFF thresholds eliminates rapid switching and extends relay and pump lifespan
constrain()Clamps the mapped value to 0–100% range — prevents out-of-range sensor readings from triggering incorrect behavior
